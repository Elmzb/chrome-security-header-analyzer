It reads. It never phones home.

The extension only looks at what your browser already received for the page you're on. It makes no extra requests, and nothing about the sites you visit is ever sent anywhere.

Install it in a minute

It's free and open-source. Here's the quick way to add it today.

1

Download it

On GitHub, click the green Code button → Download ZIP, then unzip it.

2

Open extensions

Go to chrome://extensions and switch on Developer mode (top-right).

3

Load it

Click Load unpacked and pick the unzipped folder. Pin the icon — done.
